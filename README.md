# MLBB Hub MY static website

## Run locally
Open `index.html` directly, or start a local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Files
- `index.html` — page structure
- `styles.css` — responsive layout and visual styling
- `script.js` — mobile menu and active navigation
- `assets/` — image crops made from the supplied design reference

Replace links, text, dates and images with production content before publishing.
